# WorkBalance AI - Global Solution FIAP 2025

Este é o repositório do projeto "WorkBalance AI", desenvolvido para a Global Solution da FIAP com o tema "O Futuro do Trabalho". O projeto consiste em uma solução de Front-End para um assistente inteligente de bem-estar e produtividade corporativa.
